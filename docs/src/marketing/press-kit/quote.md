# Quote — Dave, project lead

> I kept running the numbers on tracking systems for the community
> theatre rigs I work with. Follow-Me 3D was $15k. Zactrack was $60k.
> Those are the cheap options. Meanwhile a $30 USB webcam on my
> workbench could see where the beam was hitting the floor just fine.
> The math to turn pixels into pan-tilt isn't the hard part; the
> hard part is that nobody had bothered to write it.
>
> So we wrote it.

— Dave Seaman, project lead, SlyLED

# Short quote (attributable, 40 words)

> SlyLED's bet is that consumer vision hardware finally makes
> moving-head calibration a commodity problem. A $30 USB webcam
> replaces a $60k tracking system. Everything else we built stems
> from taking that seriously.

— Dave Seaman, SlyLED project lead
