AI Runtime (optional)
Local vision-language runtime (Ollama). Auto-tune defaults to the deterministic CV analyzer evaluator (no AI required); this card lets you opt into a VLM-based AI evaluator. Install only fetches Ollama itself — pull a vision model from USER_MANUAL Appendix D (e.g. ollama pull qwen2.5vl:3b) and select it below. Runs entirely on this machine — no cloud, no API keys. The orchestrator falls back to the heuristic evaluator when the VLM isn't installed.
