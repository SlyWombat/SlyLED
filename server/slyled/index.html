Supported Hardware

WiFi-connected LED performers and DMX bridges

⚡ ESP32

Up to 8 LED strings on independent GPIO pins. OTA firmware updates. Multi-pin FastLED.

📦 D1 Mini

Compact single-string controller. Perfect for portable installations. WS2812B on GPIO 2.

🛠 Arduino Giga R1

DMX-512 bridge via RS-485 transceiver. Art-Net receiver. Onboard RGB LED performer.

📡 Art-Net Nodes

Any Art-Net compatible gateway. Automatic discovery via ArtPoll. Multi-universe routing.

Get Started in Minutes

1

Download & Run

Grab SlyLED.exe or SlyLED-Setup.exe from GitHub Releases. No install required for the portable version.

2

Flash Firmware

Connect an ESP32 via USB. Go to the Firmware tab, select the port, and click Flash. Done in 30 seconds.

3

Discover & Design

Click Discover to find performers on WiFi. Position them on the 2D/3D stage. Load a preset show or build your own timeline.

4

Run the Show

Click Bake → Sync → Start. All performers execute their parts in NTP-synced unison. Watch the live emulator preview.
