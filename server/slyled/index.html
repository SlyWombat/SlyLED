Features

3D Spatial Effects

Design lighting shows in 3D space. Spheres, planes, and boxes sweep across your stage, illuminating fixtures based on their physical positions.

DMX + Art-Net Output

Control moving heads, RGB pars, and dimmers via Art-Net or sACN. Full fixture profiles with OFL library support (700+ fixtures).

LED Strip Control

WS2812B / WS2811 LED strips on ESP32 and D1 Mini. 14 built-in effects with per-pixel spatial rendering.

Timeline Editor

Multi-track timeline with clips, spatial effects, and action sequences. Bake to optimized per-fixture instructions.

Live DMX Monitor

Real-time 512-channel grid per universe. Click any channel to set values. Test fixtures with quick color buttons.

Community Profiles

Share and download DMX fixture profiles with the SlyLED community. Automatic duplicate detection.

Cross-Platform

Windows desktop app, Android companion, and ESP32/D1 Mini firmware. All connected over WiFi.

Open Source

MIT licensed. Full source on GitHub. Built with Flask, Kotlin/Compose, Arduino, and Three.js.
