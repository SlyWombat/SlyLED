Object & Person Tracking

Your lights know where people are. Automatically.

← Back to SlyLED

Point a camera at the stage. Press Track. Moving heads lock onto every person in the room and follow them in real-time — no programming, no cue lists, no operator. Add cats, dogs, chairs, or any of 16 object types and the system tracks those too. Each camera has its own detection settings so you can tune sensitivity per zone.

  • One-click tracking: press Track on the Setup tab and heads start following instantly
  • 16 detection classes — people, animals, furniture, props — choose what matters for your show
  • AI-powered: YOLOv8n runs on-camera with zero latency penalty for multiple classes
  • Per-camera tuning: adjust sensitivity, speed, and re-ID distance independently per zone
  • Smooth 40 Hz Art-Net output — heads glide, they don't jump
  • Smart assignment: 1:1 lock, round-robin cycling, or spread mode across multiple heads
  • Works in the dark — detection runs at configurable FPS with adjustable confidence
  • Temporal markers: tracked objects appear live in the 3D viewport, labeled by class
  • Fully configurable from the fixture edit dialog — no config files, no command line
  • Combine with spatial effects and timeline automation for hybrid shows
