# Stage lighting that belongs to you

Design shows in a true 3D stage model. Calibrate every moving head from a $30 USB webcam. Run the rig from your browser, a phone in your pocket, or an ESP32 puck on the wall — all of it over your own network, with none of your data leaving the building.

## What makes it different

- **Camera-assisted mover calibration** — no beacons, no IR pucks, no wands. A webcam is enough.
- **Local-first vision AI** — camera auto-tune runs on your own machine via Ollama. Cloud is optional, never required.
- **Three control surfaces, one engine** — browser, phone gyroscope, ESP32 gyro puck. Claim/release semantics keep operators from fighting over the same beam.
- **Bilingual docs, code-synchronised** — every PDF / DOCX / inline help fragment builds from the same markdown source you see in the sidebar.
- **Three-tier, open, inspectable** — read the code path from the SPA click all the way to the DMX byte.
